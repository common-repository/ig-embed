== Description ==

### Instagram Embedding               

One shortcode to embedding instagram images with full customize and unlimited colors, caption support, fully responsive and easy to use.

### The Features

* One shortcode only and easy to use [(see installation)](https://wordpress.org/plugins/ig-embed/installation/).
* Full customize and unlimited colors.
* Full caption, caption excerpt, disable caption.
* Fully responsive.
* Compatible with any lightbox.

== Installation ==

### Installation

1. Upload 'ig-embed' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

### Usage

* Just use this shortcode to embedding instagram image:
	[instagram_embedding url="https://instagram.com/p/2JqNKkABHZ/"]

### Shortcode Attributes

1. url="here enter instagram image link, default is none"
2. wrap_margin="here enter value for margin top and bottom, number only, default is 20"
3. wrap_bg="here enter color code for wrap background color, word or hex code or rgba code, example: #000000 or black or rgba(0, 0, 0, 1), default is #ffffff"
4. color"enter general color code, word or hex code or rgba code, example: #000000 or black or rgba(0, 0, 0, 1), default is #3f729b"
5. text_color="enter color code for caption text color, default is #ffffff"
6. caption="to disable caption enter: false, to display caption excerpt enter: excerpt, default is: full"
7. icon_size="enter value for instagram icon size, number only, default is 34"
8. font_size="enter value for font size, number only, default is 14"
9. before="enter link to add hyperlink before instagram image, example: https://wp-plugins.in, to enable light box enter: lightbox, default is none"
10. use s="t" to display standard instagram image, default is s="f"

### Example
	[instagram_embedding url="https://instagram.com/p/2JqNKkABHZ/" caption="excerpt" color="orange"]
	[instagram_embedding url="https://instagram.com/p/2JqNKkABHZ/" s="t"] // standard instagram image.
